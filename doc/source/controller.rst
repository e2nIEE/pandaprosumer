Controllers
===================================

.. _controllers:

In pandaprosumer, the controllers follow the
`control loop from pandapower <https://pandapower.readthedocs.io/en/latest/control/control_loop.html>`_
and are used to implement the physical model of the elements and allow to create :ref:`mappings <mappings>`
