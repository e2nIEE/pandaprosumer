###################################
Elements
###################################


A prosumer consists of an element table for each element in the prosumer. Each element table consists of a column for each
parameter and a row for each element.

pandaprosumer provides models for elements, for each of which you can find detailed
information about the definition and interpretation of the parameters in the following documentation:
